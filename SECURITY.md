# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in dual-sdk, please report it responsibly:

1. **Do not** open a public GitHub issue.
2. Email **security@blockv-labs.io** with a description of the vulnerability.
3. Include steps to reproduce if possible.

We aim to respond within 48 hours and will coordinate disclosure with you.
